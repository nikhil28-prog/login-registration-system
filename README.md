# Login and Registration System (Flask + SQLite)

This is a simple Login and Registration System built using Python, Flask, and SQLite.
It allows users to create accounts, log in securely, and log out.

## Features

* User Registration
* User Login
* Logout functionality
* Session management
* SQLite database integration
* Web-based interface using Flask

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* Git & GitHub

## How to Run the Project

1. Install Flask:

pip install flask

2. Run the application:

python login_registration_system.py

3. Open your browser and go to:

http://127.0.0.1:5000
